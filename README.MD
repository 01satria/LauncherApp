# Minimal Launcher

> *Less noise. More life.*

[![License: MIT](https://img.shields.io/badge/License-MIT-black.svg?style=flat-square)](https://opensource.org/licenses/MIT)
[![Platform: Android](https://img.shields.io/badge/Android-8.0%2B-000000.svg?style=flat-square&logo=android&logoColor=white)](#)

A home screen built on one belief — your phone should serve you, not distract you. Inspired by the quiet elegance of Nothing OS.

---

## Features

**Intelligence+ Pill Bar**  
A single adaptive bar that surfaces what matters — weather in the morning, your calendar at noon, wind-down prompts at night. No clutter. No guessing.

**Habit Heatmap**  
Track your streaks with a contribution heatmap right on the home screen. Mark habits done directly from the Pill Bar.

**Money Manager**  
Income and expense widgets that appear only when relevant. Financial clarity without leaving your home screen.

**App Drawer**  
Switch between List Mode for speed and Grid Mode for beauty. Dynamic search finds any app instantly.

---

## Design

- Dot-Matrix typography — retro soul, modern precision
- True OLED black — deeper than night, easier on battery
- Every element earns its place or disappears

---

## Installation

Requires Android 8.0 (Oreo) or higher.

```bash
git clone https://github.com/01satria/minimal-launcher
```

Open in Android Studio and run — or sideload the APK directly.
